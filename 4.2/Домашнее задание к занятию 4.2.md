## Обязательные задания

### 1. 

- В данном случае будет ошибка т.к. мы пытаемся сложить число со строкой.
```python
#!/usr/bin/env python3
a = 1
b = '2'
c = a + b
```
- Чтобы получить для переменной c значение 12
```python
#!/usr/bin/env python3
a = '1'
b = '2'
c = a + b
```
или так
```python
#!/usr/bin/env python3
a = 1
b = '2'
c = str(a) + b
```
- Чтобы получить для переменной c значение 3
```python
#!/usr/bin/env python3
a = 1
b = 2
c = a + b
```
или так
```python
#!/usr/bin/env python3
a = 1
b = '2'
c = a + int(b)
```
