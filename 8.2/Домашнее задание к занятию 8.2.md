9. https://github.com/jevgenizabelin/netology-ansible2

   В этом плейбуке выполняются 3 плея:
   Install Java, Install Elasticsearch, Install Kibana.
    
   В Install Java, идет установка java на все хосты. 
   Устанавливается переменная java_home c директорией java. 
   Присваивается тэг java. Копируется, заранее скачанный архив с определенной версией java из директории files 
   на хосты в директорию /tmp. С повышенными привилегиями, 
   проверяем директорию для установки java, разархивируем туда архив с java
   из директории /tmp. Копируем шаблон с переменными окружения java на удаленные хосты 
   и выполняем скрипт для их экспорта.
   
   В плеях Install Elasticsearch и Install Kibana скачиваем на удаленных хостах необходимые версии дистрибутивов
   в директорию /tmp. Присваиваем тэги elastic и kibana. 
   Создаем директории для установки, обьявленные как переменные elastic_home и kibana_home.
   Извлекаем архивы с дистрибутивами в директирии для установки.
   Копируем шаблоны elastic и kibana с переменными окружения
   и применяем их на хостах.
