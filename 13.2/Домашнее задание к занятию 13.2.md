##  Задание 1: подключить для тестового конфига общую папку

- front-back.yaml

```
apiVersion: apps/v1
kind: Deployment
metadata:
  name: front-back
  namespace: stage
  labels:
    app: front-back
spec:
  replicas: 1
  selector:
    matchLabels:
      app: front-back
  template:
    metadata:
      labels:
        app: front-back
    spec:
      containers:
        - name: frontend
          image: jay15/13-frontend
          imagePullPolicy: IfNotPresent
          ports:
          - containerPort: 80
          env:
          - name: BASE_URL
            value: http://localhost:9000
          volumeMounts:
            - mountPath: /static
              name: share-folder
        - name: backend
          image: jay15/13-backend
          imagePullPolicy: IfNotPresent
          ports:
          - containerPort: 9000

            - mountPath: /static
              name: share-folder

      volumes:
        - name: share-folder

---

    - name: back
      protocol: TCP
      port: 9000
      targetPort: 9000

```

      resources:
        requests:
